    Esta versão, tal como a Version1 apenas funcionam para Linux (devido a cor
        de texto), se preferir, para melhor melhor resolução execute a versão
        de Windows em qualquer versão desde o Windows 7

    Para compilar o programa, introduza no terminal: javac Connect4GUI.java
    Para executar o programa, inroduza no terminal: java Connect4GUI
    

    Estes programas foram criados e executados em máquina virtual Ubuntu
18.04.1 LTS com 3,9 GB de RAM, Intel Core i7-8550U CPU @ 1.80GHz x 4 fornecidos.


    As versões de Java e do compilador são:
        OpenJDK "10.0.2" 2018-07-17
        Javac 10.0.2
