    Esta versão foi criada para Windows, funciona para versões a partir do 7

    Para compilar o programa, introduza no terminal: javac Connect4GUI.java
    Para executar o programa, inroduza no terminal: java Connect4GUI

    Note, é necessário ter o java instalado e JRE também.
    

    Estes programas foram criados e executados em máquina  64 bits Windows 10 
Home com 16 GB de RAM, Intel Core i7-8550U CPU @ 1.80GHz 1.99 GHz 8 cores


    As versões de Java e do compilador são:
        OpenJDK "10.0.2" 2018-07-17
        Javac 9.0.4
