    Ao executar o programa, deverá escolher qual a opção:
        1)MinMax no pruning
        2)MinMax alpha-bet pruning
        3)Monte Carlo Tree Search
        4)Exit
    Deverá introduzir um número entre os valores e primir Enter, depois será
        levado para o mode de jogo que escolheu, em cada um, deverá fornecer
        um limite de profundidade e depois escolher qual o jogador inicial.
    
    Durante o jogo, basta colocar a coluna onde quer que a peça caia. Quando 
        o jogo terminar, será informado.
        

    Para compilar o programa, introduza no terminal: javac ConnectFour.java
    Para executar o programa, inroduza no terminal: java ConnectFour
    

    Estes programas foram criados e executados em máquina virtual Ubuntu
18.04.1 LTS com 3,9 GB de RAM, Intel Core i7-8550U CPU @ 1.80GHz x 4 fornecidos.


    As versões de Java e do compilador são:
        OpenJDK "10.0.2" 2018-07-17
        Javac 10.0.2
